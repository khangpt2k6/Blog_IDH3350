This diagram maps the Kollmuss & Agyeman (2002) model of pro-environmental behavior — the full chain from Knowledge → Awareness → Attitude → Behavior. Each barrier icon shows where the chain typically breaks. Use +/− to zoom, or scroll inside the diagram. Start with the red barrier — the Green Gap.
